# Axis - v0.2.0-beta.0 (#2.2607201802)  [[Download](https://github.com/xYuri/Axis-App/releases/tag/0.2.0-beta.0)]

---

---

## Release Notes:

---

### 🆕 Added
- Tabs in `Weight Monitor` page for better organizing and control
- Tab `Overview` in `Weight Monitor` module
- `Overview` tab's UI page in `Weight Monitor` module
- Tab `Week` in `Weight Monitor` module
- `Week` tab's UI page in `Weight Monitor` module
- Tab `Month` in `Weight Monitor` (Tab `Month` remains under construction for now)
- Improve the `Weight Monitor` chart layout
- Improve the `Weight Monitor` chart animations
- Option `Animate Navigation Drawer` in `Setting` -> `Appearance`
- Groups to `Navigation Drawer` for better organization and visibility
- Release note for the old version `0.1.0`

---

### 🪲 Fixed
- A visual bug caused the `Weight Monitor` to seem that it doesn't save old weight, caused by reverse ordering of the dates

---

### 🔃 Updated
- Rework `Weight Monitor` page
- `Settings` page no longer reloads when changing theme, keeping the state, scroll position, and expanded/collapsed foldables
- Rework the whole `Navigation` system, improving performance and adding internal features
- Rework the app to use the new `Navigation` system
- Improve `NavDrawer` expand/collapse events handling and performance
- Increase the speed of which the `NavDrawer` expand/collapse (from 1000ms to 500ms)
- Add color to the `Navigation Drawer` expand icon for better visibility
- Move `Navigation Drawer` expand icon and app name to the `Title Bar` on `Desktop` platforms
